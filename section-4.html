4 Typografi

Vi använder oss av Open Sans från Google Fonts. Vid behov av att använda ytterligare typsnitt ska dessa väljas med hänsyn till tillgänglighet och öppenhet.

4.1 Storlekar

Element Storlek Vikt
h1 36px 700
h2 30px 700
h3 24px 700
h4 20px 700
p 16px 400

4.1.1 Rubriker

h1 - This is a main title

h2 - This is a subtitle

h3 - This is a section title

h4 - This is a section subtitle

<h1>h1 - This is a main title</h1>
<h2>h2 - This is a subtitle</h2>
<h3>h3 - This is a section title</h3>
<h4>h4 - This is a section subtitle</h4>

4.1.2 Sekundär text / Liten text

Implemented using <small> or .small

h1 - This is a main title - Secondary text

h2 - This is a subtitle - Secondary text

h3 - This is a section title - Secondary text

h4 - This is a section subtitle - Secondary text

p - This is a normal text with secondary text inside of it.

<h1>h1 - This is a main title - <small>Secondary text</small></h1>
<h2>h2 - This is a subtitle - <small>Secondary text</small></h2>
<h3>h3 - This is a section title - <small>Secondary text</small></h3>
<h4>h4 - This is a section subtitle - <small>Secondary text</small></h4>
<p>p - This is a normal text with <small>secondary text</small> inside of it.</p>

This text contains an inserted word.

4.3.4 Fet text

This text contains an important word.

This text contains a bold word.

<p>This text contains an <strong>important</strong> word.</p>
<p>This text contains a <b>bold</b> word.</p>

4.3.5 Kursiv / Betonad

This text contains an italicized word.

<p>This text contains an <i>italicized</i> word.</p>

4.3.8 Förkortningar / initialism

An abbreviation of the word attribute is attr.

JSON-LD is pretty cool.

<p><abbr title="JavaScript Object Notation for Linked Data" class="initialism">JSON-LD</abbr> is pretty cool.</p>
